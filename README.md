# Blog Content Management System

## Description

The **Blog Content Management System (CMS)** is a web-based application that allows users to create, manage, and display blog posts dynamically. The system is designed to provide a simple platform where blog content can be published and organized efficiently.

This project demonstrates the implementation of a full-stack web application using frontend and backend technologies. It allows users to add blog posts, upload images, categorize content, and interact through comments.

## Features

* Create and publish blog posts
* Upload images for blog posts
* Display blog posts dynamically
* Category management
* Comment system for user interaction
* Responsive user interface

## Technologies Used

* **HTML** – Structure of the web pages
* **CSS** – Styling and layout design
* **JavaScript** – Client-side interactivity
* **PHP** – Server-side scripting
* **MySQL** – Database management

## Installation / Setup

Follow these steps to run the project locally:

1. Clone the repository:
   github https://github.com/24amtics009-spec/upskillcampus.git

2. Move the project folder to your local web server directory (for example, `htdocs` if using XAMPP).

3. Import the provided database file into MySQL using phpMyAdmin.

4. Start **Apache** and **MySQL** services in your local server environment.

5. Open your browser and run the project:
   http://localhost/blog-cms

## Project Structure

* **/css** – Stylesheets for page design
* **/js** – JavaScript files for interactivity
* **/images** – Uploaded images for blog posts
* **/php** – Backend scripts for server operations
* **index.php** – Main page displaying blog posts
